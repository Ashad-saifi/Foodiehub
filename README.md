# 🍔 FoodieHub – Smart Food Ordering Platform

FoodieHub is a modern, responsive **food ordering web application** inspired by platforms like Zomato. It provides users with a seamless experience to explore restaurants, browse menus, and order food online with an attractive and user-friendly interface.

---

## 🚀 Features

* 🔍 **Search & Discover** restaurants easily
* 🍽️ **Browse Menus** with detailed food items
* 🛒 **Add to Cart** functionality
* ❤️ **Wishlist / Favorites** option
* 📱 **Responsive Design** (Mobile + Desktop)
* 🎨 **Modern UI/UX** inspired by real-world apps
* ⚡ Fast and smooth performance
